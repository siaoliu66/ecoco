# 報價單產生器 (Quote Generator)

純前端報價單工具，左側輸入資料、右側即時預覽，支援 PDF 下載、多幣別、折扣與稅金計算、歷史報價儲存。

## 使用方式
直接開啟 `index.html`，或部署到 GitHub Pages 後透過瀏覽器使用，無需安裝、無需伺服器。

## 功能
- 新增/刪除產品項目
- 品名、數量、單價
- 自動計算小計/折扣/稅金/總額
- 即時報價單預覽
- PDF 下載
- 5% / 8% 稅金切換
- 客戶資料欄位
- 套用公司 Logo 與主題顏色
- 儲存與載入歷史報價（瀏覽器本機儲存）
- TWD / USD 幣別切換
- 折扣計算（百分比或固定金額）
